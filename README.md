# Kedai Kopi Kenangan Senja ☕🌇

Website sederhana untuk Kedai Kopi Kenangan Senja, dibuat sebagai latihan dan media belajar membangun website statis.

---

## 🛠 Teknologi yang Digunakan

Project ini dibuat menggunakan 3 bahasa pemrograman utama:

- **HTML** - Untuk struktur dan isi konten website
- **CSS** - Untuk mempercantik tampilan dan style website
- **JavaScript (JS)** - Untuk menambahkan interaktivitas sederhana

---

## 💻 Fitur Website

✔ Landing page sederhana  
✔ Halaman Tentang Kami  
✔ Navigasi menu  
✔ Responsive layout (tampilan menyesuaikan ukuran layar)  
✔ Footer dengan link ke media sosial
